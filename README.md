# TODO 管理器 - 功能完整版

## 项目介绍
这是一个基于 Python 开发的命令行交互式 TODO 任务管理器，支持任务的增删改查、状态切换、持久化存储、导入导出、排序等核心功能，所有任务数据以 JSON 格式本地保存，操作简单、功能完整，适合日常任务管理练习和使用。

## 运行环境
- Python 3.6 及以上版本（依赖 Python 内置库，无需额外安装第三方包）
- 支持 Windows/macOS/Linux 所有主流操作系统

## 核心功能列表
| 功能编号 | 功能名称               | 功能描述                                   |
|----------|------------------------|--------------------------------------------|
| 1        | 添加任务               | 输入任务内容，自动记录创建时间，任务默认未完成 |
| 2        | 查看所有任务           | 展示所有任务（含序号、状态、创建时间），并显示统计信息 |
| 3        | 删除任务               | 按任务序号删除指定任务                     |
| 4        | 修改任务               | 按序号修改任务内容，自动记录修改时间       |
| 5        | 切换任务状态           | 切换指定任务的“完成/未完成”状态            |
| 6        | 搜索任务               | 按关键词模糊搜索任务（不区分大小写）       |
| 7        | 保存到文件             | 将任务列表手动保存到 `tasks.json` 文件     |
| 8        | 从文件读取             | 从 `tasks.json` 加载已保存的任务列表       |
| 9        | 清除已完成的任务       | 批量删除所有标记为“已完成”的任务（需确认） |
| 10       | 排序任务               | 支持3种排序方式：创建时间、内容字母序、任务状态 |
| 11       | 导出任务到文本         | 将任务列表导出为 TXT 格式文本文件          |
| 12       | 导入任务从文本         | 从 TXT 文本文件导入任务（兼容导出的格式）  |
| 0        | 退出程序               | 退出前自动保存任务，避免数据丢失           |

## 使用方法

### 步骤 1：保存代码
将完整代码保存为 `todo_manager.py`（文件名可自定义）。

### 步骤 2：运行程序
打开终端/命令提示符，进入代码所在目录，执行以下命令：
```bash
python todo_manager.py
